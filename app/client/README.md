# Agent HQ Client

React frontend for Agent HQ.

## Setup

```bash
cd app/client
npm install
```

## Running

```bash
npm start
```

Client runs at http://localhost:3000

## Scripts

```bash
npm start       # Development server
npm run build   # Production build
npm test        # Run tests
```
